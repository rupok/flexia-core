== Description ==

Core plugin for Flexia theme. Controls all the plugin territory functionalities. 

== Installation ==

1. Upload `flexia-core.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress\

== Frequently Asked Questions ==

= Does it work standalone =

Though it's a specialized plugin for flexia, it will work standalone as well.
